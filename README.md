Консольное приложение:
-пользователь вводит логин и пароль. Происходит чтение из файла txt.txt. Если кошелек с этим юзерм уже есть, загружается кошелек из файла и редактируется уже он. Если кошелька нет, создается новый;
-пользователь вводит планируемый бюджет с названием категории;
-ввод источников и сумм дохода;
-ввод расходов с названием категорий
Если названия категорий в бюджете и расходах совпадают, это отражается на остатке планируемого бюджета
После окончания работы, данные кошелька записываются в /resources/txt.txt
