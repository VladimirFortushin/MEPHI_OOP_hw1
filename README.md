Консольное приложение:
-пользователь вводит планируемый бюджет с названием категории;
-ввод источников и сумм дохода;
-ввод расходов с названием категорий
Если названия категорий в бюджете и расходах совпадают, это отражается на остатке планируемого бюджета
После окончания работы, данные кошелька записываются в /resources/txt.txt

Пример работы:
Введите имя пользователя: XÆA-12
Введите пароль: 456
Спланируйте бюджет (прим. Развлечения:30000): Еда:1000
Добавлен бюджет: Еда 1000

Отразите доходы (прим. Зарплата:80000): Еда:500
Добавлен доход: Еда 500

Отразите расходы (прим. Кафе:1500): Еда:180
Добавлен расход: Еда 180
Еда:200
Добавлен расход: Еда 200

Завершение.
Кошелек пользователя:
Пользователь: XÆA-12,
Общий баланс: 120.0,
Доходы:
  Еда: 500.0
Расходы:
  Еда: 380.0
Бюджетный план:
  Еда: 1000.0 остаток: 620.0

Данные кошелька сохранены в src\main\resources\txt.txt
